# Dashboard de Posgrados

Este proyecto es una plantilla de dashboard web para mostrar información de tres programas de posgrado: Doctorado, Maestría y Médico Quirúrgica.

## Qué incluye
- Diseño limpio y responsive.
- Tres secciones separadas para cada programa.
- Gráficas con Chart.js.
- Estructura lista para publicar en Netlify.

## Cómo actualizarlo con datos del Excel web
1. Exporta tus datos desde el archivo de Excel a CSV o JSON.
2. Reemplaza el contenido de `data.json` con los datos actualizados.
3. Si quieres que el sitio se alimente automáticamente, puedes conectar el archivo a una automatización externa (Power Automate, Google Apps Script o un backend) que publique un JSON actualizado.

## Publicar en Netlify
1. Sube esta carpeta a GitHub o GitLab.
2. En Netlify, crea un nuevo sitio desde ese repositorio.
3. El sitio se publicará automáticamente.
